# INTEGRA — Website Copy & UX Structure [v2.0 - Premium Edition]

## 1. Configuración de Estilo (Arquitectura Visual)
- **Paleta de Colores:** - **Primario (Autoridad):** `#4A4063` (Púrpura profundo) para CTAs, fondos de impacto y acentos técnicos.
  - **Secundario (Soporte):** `#C8C6D7` (Lavanda grisáceo) para bordes de 1px, placeholders y etiquetas Mono.
  - **Fondo:** `#F5F5F5` (Blanco roto) para secciones de lectura limpia.
  - **Texto:** `#0E0E0E` (Negro carbón) para máxima legibilidad.
- **Tipografía:** - Títulos y cuerpo: `Inter` (Sans Serif técnica).
  - Datos y KPIs: `JetBrains Mono` (Estética de ingeniería).
- **Layout:** Grid de 8pt, ancho máximo 1200px, secciones con aire visual amplio.

### SECCIÓN 3: PROPUESTA DE VALOR (Sistemas Reales)
*Uso de `#4A4063` para los números de sección.*

#### 01. Auditoría Operativa y Financiera
- Analizamos el negocio como un sistema integral.
- Visibilidad real de P&L y flujos de caja operativos.
- **Resultado:** Claridad brutal sobre la salud del negocio.

#### 02. Arquitectura de Sistemas
- Procesos lean y estandarización técnica.
- Integración de arquitectura digital útil (n8n, AWS, ERPs).
- **Resultado:** Ejecución que no depende de "héroes" diarios.

#### 03. Ejecución y Acompañamiento
- Implementación directa junto a líderes y equipos.
- Ajustes en tiempo real sobre datos operativos reales.
- **Resultado:** Hábitos operativos nuevos y rentabilidad sostenible.
